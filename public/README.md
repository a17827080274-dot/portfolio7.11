# Public Directory / 公共目录

Here is where you can upload your work display images.
请在这里上传您的作品展示图片：

- **第一张：编排海报的展示图**
  Filename: `编排海报设计.png` (or `.jpg`, `.jpeg`, `.webp`)
  Path in app: `/公共/编排海报设计.png

- **第二张：AI公开赛的展示图**
  Filename: `ai公开赛.png` (or `.jpg`, `.jpeg`, `.webp`)
  Path in app: `/ai公开赛.png`

- **第三张：思辨设计的展示图**
  Filename: `思辨设计.png` (or `.jpg`, `.jpeg`, `.webp`)
  Path in app: `/思辨设计.png`

- **第四张：其他设计的展示图**
  Filename: `其他设计.png` (or `.jpg`, `.jpeg`, `.webp`)
  Path in app: `/其他设计.png`

---
These files will be automatically loaded by the corresponding memory card detail sections inside the app.
这些文件会被应用里的对应内存卡详情页自动加载和完美呈现。
